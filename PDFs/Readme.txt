Diretório onde serão colocados os contos em formato PDF.


Percebi que o link para eles é um pouco diferente. Para que possam ser baixados,
abra o arquivo clicando nele dentro do Github e copie o link na barra do navegador.

Depois disso, faça uma modificação no link. Veja o exemplo abaixo, com o arquivo
A Vontade do Universo no diretório PDFs:

A Vontade do Universo
Link que aparece na barra do navegador:
https://raw.github.com/psicspsix/LeitorPDFparaSmartTVs/blob/main/PDFs/A%20Vontade%20do%20Universo.pdf

Modificação que você deve fazer:
https://raw.githubusercontent.com/psicspsix/LeitorPDFparaSmartTVs/main/PDFs/A%20Vontade%20do%20Universo.pdf

Diversas formas de abrir as páginas foram testadas: Visão em 100%, automatica, page width e page fit.
Mas nenhuma funcionou nas Smart TVs. Foram testadas as marcas Samsung e LG.
